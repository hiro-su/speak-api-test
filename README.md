# QRコード 音声読み上げシステム（データ圧縮版）

スマートフォンの標準カメラでQRコードを読み取るだけで、ブラウザに内蔵されたWeb Speech APIを利用してテキストを音声で読み上げる、サーバーレスの音声ガイドシステムです。

## デモ
以下のリンクからシステムの動作を確認できます。
👉 **[デモページ（QRコード作成・読み上げ）](https://hiro-su.github.io/speak-api-test/)**

## 概要と特徴

QRコードに長文の日本語テキストを埋め込み、URLのパラメータとして受け渡すアプローチを採用しています。

*   **サーバーレス（スタンドアローン）:** テキストデータはすべてQRコード内（URLパラメータ）に保持されるため、データベース等のバックエンドサーバーは一切不要です。
*   **LZ-Stringによる高圧縮:** 日本語テキストをそのままURLパラメータにすると、URLエンコードによりデータ量が約9倍に膨れ上がります。このシステムでは、JavaScriptの圧縮ライブラリ「LZ-String」を利用してテキストを高効率に圧縮することで、500文字〜600文字程度の日本語であっても、スマートフォンの標準カメラで読み取り可能な現実的な長さのURL（1000〜1500文字程度）に抑えています。
*   **Androidの音声停止バグ対策:** Android端末において、長文の読み上げが途中で停止してしまう既知のバグを回避するため、文章を「。」などの句点で分割して連続再生する処理を実装しています。

## 使い方

1.  **QRコードの生成**
    *   デモページにアクセスします。
    *   画面下部の「【生成用】QRコード作成ツール」のテキストエリアに、読み上げさせたい文章（500〜600文字程度を推奨）を入力します。
    *   「圧縮してQRコードを生成」ボタンを押すと、圧縮済みのテキストがパラメータとして付与されたURLのQRコード画像が生成されます。
2.  **QRコードの読み取りと再生**
    *   生成されたQRコードを、スマートフォンの標準カメラ（iOSのカメラアプリ、AndroidのGoogleレンズ等）で読み取ります。
    *   ブラウザ（SafariやChrome）が立ち上がり、読み取り専用の画面が表示されます。
    *   「▶ 音声を再生する」ボタンをタップすると、テキストが合成音声で読み上げられます。
